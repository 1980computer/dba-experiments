Day 27: Color Thief

Utils.randomImage() and PageComponent

Utilizing the Color Thief module created by Marc Krenn, this small demo showcases the rgb value of the dominant color in each photo. Photos are retrieved from Unsplash using Utils.randomImage(), a handy function that I previously didn't know existed. For some reason though the colorThief.getColor function doesn't work with loops, so I hard coded the color assignments.
